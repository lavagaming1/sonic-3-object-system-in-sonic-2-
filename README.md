# Lavagamings Sonic 2 Disassembly 
This is a Sonic 2 disassembly with a small port of the Sonic 3 object system to Sonic 2, with a few edits on the object manager by RandomName.

The whole game has been converted to use the Sonic 3 object system, so you may run into a few crashes and bugs.

If you happen to want to know the current fixes or you know how to help make this better, please visit [the Team Conquest Discord Server!](https://discord.gg/zr4n2GdTfJ)
